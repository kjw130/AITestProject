# Project Name

## Stack
- PHP 8.x
- SQLite (via PDO)
- Vanilla JavaScript (ES6+)

## Setup
1. Clone this repo
2. Make sure PHP is installed (`php -v`)
3. Make sure SQLite is enabled (`php -m | grep sqlite`)
4. Run locally: `php -S localhost:8000 -t public`
5. Open browser at `http://localhost:8000`

## Folder Structure
/public      → Entry point and assets
/src         → PHP classes and logic
/db          → SQLite database
/tests       → Test files
